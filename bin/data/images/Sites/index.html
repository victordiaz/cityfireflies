Tu sitio web.

Crea y publica tu propio sitio web de manera rápida y fácil con iWeb, Pages y muchas otras aplicaciones disponibles para Mac OS X.

Es un instante crear y publicar tu propio sitio web desde tu Mac. Cuando tu web esté listo, publicarlo es igual de sencillo.

Abre Preferencias del Sistema, haz clic en Compartir y selecciona Compartir Web.

Ya está. Tu sitio web está ahora disponible en tu red privada en casa o en el trabajo.

Si estás conectado a Internet, tu web también puede estar disponible para tus amigos estén donde estén. Solo tienes que enviarles la dirección indicada en el panel de preferencias Compartir.

La potencia de Apache

Compartir Web está integrado en el servidor web Apache, una tecnología estándar en el sector incluida en Mac OS X. Para obtener más información sobre el servidor web Apache, consulta el manual de Apache.
